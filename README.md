=================

Grunt task to strip or add BOM to files

```shell
npm install grunt-contrib-bom --save-dev
```

```js
grunt.loadNpmTasks('grunt-contrib-clean');
```

```js
bom: {
  src: 'test/tmp/add/*.txt'
}
```

Lets go dancing!
